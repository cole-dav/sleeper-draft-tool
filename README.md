# Sleeper Draft Board

Live site: [sleeper-draft.netlify.app](https://sleeper-draft.netlify.app/)

A web app for visualizing and managing Sleeper league draft boards, including team needs, pick tracking, and customizable team order.

## Quick Start

```bash
npm install
npm run dev
```

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run check
npm run db:push
```

## Environment

Set `DATABASE_URL` for the Postgres connection string. For Netlify Functions, use the Supabase pooler (port 6543).
